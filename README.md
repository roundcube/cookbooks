 * `vagrant up`

## Server prep

The server should run Ubuntu Precise. Before starting, please ensure the latest updates have been applied.

 * `apt-get update`
 * `apt-get upgrade`
 * `apt-get dist-upgrade`
 * `apt-get install ruby1.9.3`

### Ruby setup

 * create `~/.gemrc`
 * paste this into it: `gem: --no-ri --no-rdoc`
